# dotcam

Created by Yofre (yfrvids)

dotcam es un script que usa ffplay (ffmpeg)

## Script

- Copiar y pegar el archivo dotcam en /usr/local/bin/ (`sudo cp -r dotcam /usr/local/bin/`)
- Darle permisos al archivo de script (`sudo chmod +x /usr/local/bin/dotcam`)
- Ejecutar mediante el comando `dotcam`
